# Воркшоп
[Владимир Бурмистров](https://t.me/CrazyElephant_note) из [Techanalyst club](https://t.me/techanalyst_club) провел Воркшоп  Мониторинг и логирование — Kibana, Grafana, ElasticSearch

Материалы к воркшопу.

## Описание

> **Аудитория:** Системные аналитики (уровень middle/senior)  
> **Формат:** Практический воркшоп, 4–6 часов  
> **Результат для участника:** Работающий стек мониторинга на локальной машине, понимание трёх столпов observability, навык чтения дашбордов и логов, знание ключевых фреймворков

Воркшоп построен по принципу **"теория → демо → практика"** в коротких блоках по 30–45 минут. Каждый блок заканчивается конкретным артефактом: работающим контейнером, настроенным datasource или готовым дашбордом. Инструментальная база взята из проекта **OneClickMonitoring** и расширена сценариями для системных аналитиков.

Три столпа observability — **Logs, Metrics, Traces** — являются основой современного мониторинга распределённых систем. Воркшоп покрывает первые два (логи и метрики) глубоко на практике; трейсинг разбирается теоретически как следующий шаг.
